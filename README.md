وال اضافه‌کردن یک خدمت جدید (الگوی نظام وظیفه)

برای هر سرویس جدید که مثل «معافیت پزشکی / کفالت» باشد:

سرویس دیتا
services/<service>.js
→ تعریف window.SERVICE_DATA.<key> + meta.feeKey

HTML سرویس
<service>.html
→ کپی دقیق پزشکی
→ فقط تغییر SERVICE_KEY و مسیر فایل سرویس

هزینه‌ها
fees.js
→ افزودن کلید هزینه مطابق meta.feeKey

هاب نظام وظیفه
military-hub.html
→ فعال‌کردن کارت و لینک به <service>.html

سرچ لندینگ
services/catalog.js
→ افزودن آیتم با href
→ اگر مثل پزشکی نباید در All بیاید: hiddenFromAll:true

تمام.
اگر این ۵ مرحله انجام شود، سرویس:

از هاب می‌آید

در سرچ پیدا می‌شود

در All نمایش داده نمی‌شود

هزینه‌ها درست رندر می‌شوند
